## ğŸ¯ Quick Reference

### Most Common Git Commands

```bash
# Initialize repository
git init

# Check status
git status

# Add files
git add .

# Commit changes
git commit -m "Your message"

# Push to remote
git push origin main

# View commit history
git log --oneline

# Fix last commit
git commit --amend -m "New message"

# Go back to previous commit
git reset --hard <commit-hash>
```
